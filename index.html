langkah-langkah Instalasi Pop!_OS Dual Boot dengan Windows 11

  • Prosedur Instalasi
  • Tentang Pop!_OS

Panduan ini akan memandu Anda langkah demi langkah dalam menginstal Distro Linux Pop!_OS sebagai sistem operasi dual boot (berdampingan) dengan Windows 11. Panduan ini berdasarkan video YouTube dari channel Levelup iD: WINDOWS 11 HATI HATI! Cobain POP OS Distro Linux Yang Keren dan LENGKAP DUALBOOT.

Pastikan Anda sudah mencadangkan data penting Anda sebelum memulai!

1. Persiapan (Unduh Bahan)

Langkah pertama adalah mengunduh file ISO Pop!_OS dan aplikasi untuk membuat bootable USB drive.

1.1. Unduh Pop!_OS

  • Kunjungi situs web resmi Pop!_OS: pop.system76.com.
  • Pilih versi yang sesuai dengan GPU Anda (ada versi khusus untuk Nvidia jika Anda memiliki kartu grafis Nvidia).

1.2. Unduh Aplikasi Flashing (Balena Etcher)

  • Unduh Balena Etcher dari situs resminya (balena.io/etcher) atau gunakan Rufus sebagai alternatif.

2. Membuat Partisi Kosong di Windows

Anda perlu menyiapkan ruang kosong di hard drive Anda untuk instalasi Pop!_OS.

  • Klik kanan pada ikon menu Start Windows, lalu pilih *Disk Management*.
  • Pilih drive yang memiliki banyak ruang kosong (misalnya C: atau D:).
  • Klik kanan pada drive tersebut, lalu pilih *"Shrink Volume..."*.
  • Tentukan ukuran ruang yang ingin Anda alokasikan untuk Pop!_OS (misalnya 150000 MB untuk 150GB).
  • Klik *"Shrink"*. Ini akan membuat "Unallocated space".

3. Membuat Bootable USB Drive

Gunakan Balena Etcher untuk mentransfer file ISO Pop!_OS ke flash drive Anda.

  • Pasang flash drive Anda (minimal 8GB).
  • Buka Balena Etcher.
  • Klik *"Flash from file"* dan pilih file ISO Pop!_OS yang sudah Anda unduh.
  • Klik *"Select target"* dan pilih flash drive Anda.
  • Klik *"Flash!"* dan tunggu prosesnya selesai.

4. Proses Instalasi Pop!_OS

4.1. Masuk ke BIOS/UEFI dan Atur Boot Priority

  • Restart PC Anda.
  • Tekan tombol yang sesuai (biasanya F2, F10, F12, atau Del) berulang kali saat PC menyala untuk masuk ke menu BIOS/UEFI.
  • Di dalam BIOS/UEFI, navigasikan ke bagian *Boot Options* atau *Boot Priority*.
  • Ubah urutan boot agar *UEFI USB Drive* Anda menjadi pilihan pertama.
  • Simpan perubahan dan keluar dari BIOS/UEFI.

4.2. Memulai Instalasi Pop!_OS

  • PC akan boot dari USB drive Anda. Ikuti petunjuk di layar untuk memilih bahasa dan layout keyboard.
  • Saat tiba di layar "How would you like to install?", pilih *"Custom (Advanced)"*. Ini sangat penting untuk dual boot.

4.3. Membuat Partisi Linux

Pada ruang "Unallocated" yang telah Anda buat sebelumnya, Anda akan membuat 4 partisi baru:

  1. Partisi EFI (Boot)
    • Ukuran: 1GB
    • Tipe: *FAT32*
    • Gunakan sebagai: *"Boot / EFI System Partition"*
  2. Partisi Swap
    • Ukuran: Setidaknya sama dengan RAM Anda, atau sekitar 8GB.
    • Tipe: *Linux Swap*
  3. Partisi Root (/)
    • Ukuran: Minimal 30GB-50GB (contoh di video 50GB).
    • Tipe: *Ext4*
    • Gunakan sebagai: *"/" (root)*
  4. Partisi Home (/home)
    • Ukuran: Gunakan *sisa ruang* yang tersedia (misalnya sekitar 90GB).
    • Tipe: *Ext4*
    • Gunakan sebagai: *"/home"*

Setelah partisi dibuat dan diatur, lanjutkan proses instalasi. Buat akun pengguna dan password Anda.

5. Pengaturan Dual Boot (GRUB Customizer)

Setelah instalasi selesai, PC mungkin akan langsung boot ke Pop!_OS. Anda perlu mengatur agar muncul menu pilihan saat startup.

  • Boot ke Pop!_OS yang sudah terinstal.
  • Buka aplikasi Terminal.
  • Jalankan perintah berikut satu per satu untuk menginstal os-prober dan grub-customizer (tekan Enter setelah setiap baris dan masukkan password jika diminta):
    • sudo apt update
sudo apt install os-prober
sudo os-prober
sudo update-grub
sudo add-apt-repository ppa:danielrichter2007/grub-customizer
sudo apt update
sudo apt install grub-customizer
  • Setelah instalasi grub-customizer selesai, buka aplikasinya (cari "Grub Customizer" di menu aplikasi).
  • Di Grub Customizer, Anda bisa mengatur waktu tunggu menu boot (misalnya 10 detik) dan memastikan entri Windows Boot Manager terlihat. Simpan perubahan.
  • Restart PC Anda. Kembali masuk ke *BIOS/UEFI*.
  • Di bagian *Boot Options, pastikan **"Pop!_OS"* (atau "ubuntu" jika Pop!_OS tidak eksplisit) adalah pilihan boot pertama. Simpan dan keluar.

Sekarang, setiap kali Anda menyalakan PC, Anda akan melihat menu yang memungkinkan Anda memilih untuk boot ke Pop!_OS atau Windows Boot Manager.

Selamat! Anda telah berhasil menginstal Pop!_OS dalam konfigurasi dual boot dengan Windows 11.
